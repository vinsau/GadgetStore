Running the Program
--------------------
Using VS Code:

1. Download both files
2. Open the terminal (Ctrl + ~).
3. Navigate to the program's directory/folder using:

'cd /path/to/directory'

4. Compile the program:

'g++ -o GadgetStoreSingle GadgetStoreSingle.cpp'

5. Run the program:

'./GadgetStoreSingle'
